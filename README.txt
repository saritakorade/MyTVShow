The Application is Developed in PHP. Its easy to setup application using PHP. We need only XAMPP server to run the Application using PHP.


1. Download the XAMPP from "https://www.apachefriends.org/download.html".

2. Run the XAMPP server.

3. Open XAMPP control Panel. Start Apache.(Check the port. there may be port conflict. Default port is 80.)

4. Go to C:\\xampp\htdocs

5. Unzip the MyTVShow folder in htdocs.

6. Open the browser.
 
7. Goto http://localhost/MyTVShow/index.php

8. Application will be running. Here we go..
